How to use

Create an yapp element by adding data-yapp-img attribute


    <div data-yapp-img="images/image.jpeg"></div>

Add script before closing </body>


    <script src="js/yapp.min.js"></script>

Options

Set a custom height by using data-yapp-height attribute. Default height is set to 1/3rd viewport height.


    <div data-yapp-img="images/image.jpeg" data-yapp-height="300px"></div>

Set a custom width by using data-yapp-width attribute. Margin auto is added by default to center yapp element.


    <div data-yapp-img="images/image.jpeg" data-yapp-width="80rem"></div>

Prevent image from scrolling by adding data-yapp-static attribute.


    <div data-yapp-img="images/image.jpeg" data-yapp-width="80rem" data-yapp-static></div>
